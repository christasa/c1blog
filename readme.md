# C1blog



## 使用方法

在文件目录下执行：
```
pip install -r requirements.txt

python manage.py makemigrations

python manage.py migrate
```

##### 创建管理员账号：
```
python manege.py createsuperuser
```
##### 打开本地服务器：
```
python manege.py runserver 9000
```
浏览器打开 `主机ip:9000` 查看效果
